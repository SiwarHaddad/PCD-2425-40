Frequently Asked Questions

What image formats does FIDS support?

expand_more

FIDS supports all common image formats including JPEG, PNG, TIFF, BMP, RAW, and many others. For best results, we recommend using uncompressed or minimally compressed formats to preserve image details that are crucial for forensic analysis.

How long does the analysis process take?

expand_more

The AI analysis is typically completed within minutes. Expert review timing depends on case complexity and queue status, but is usually completed within 24-48 hours. Rush processing is available for time-sensitive cases.

Is my data secure during the analysis process?

expand_more

Yes, FIDS employs end-to-end encryption for all uploads and stored data. Our system is compliant with major security standards including GDPR, HIPAA, and SOC 2. All experts sign confidentiality agreements and access is strictly controlled.

Can I export the analysis reports?

expand_more

Yes, all analysis reports can be exported in multiple formats including PDF, DOCX, and HTML. These reports are designed to be court-admissible and include all necessary methodology documentation.

How many users can I add to my account?

expand_more

The number of users depends on your subscription plan. Basic plans include 1 user, Professional plans include up to 5 users, and Enterprise plans offer unlimited users. Additional user licenses can be purchased for any plan.
